# Security Policy

## Supported Versions

The following versions of Devsetup are currently supported with security updates.

| Version | Supported |
|--------|-----------|
| 1.x | ✅ |
| < 1.0 | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability in Devsetup, please report it responsibly.

Do NOT open a public issue for security vulnerabilities.

Instead:

1. Open a private security report through GitHub Security Advisories
2. Or contact the maintainer through GitHub

We will investigate and fix the issue as soon as possible.

## Security Best Practices

When using Devsetup:

• Do not run commands from untrusted repositories  
• Always review detected install commands before executing them  
• Keep your Devsetup version updated
